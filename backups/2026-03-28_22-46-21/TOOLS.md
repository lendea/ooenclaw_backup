# TOOLS.md - 本地笔记

Skills 定义工具_如何_工作。这个文件是为_你的_具体细节 —— 那些你设置中独有的东西。

## 这里放什么

像这样的东西：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好的语音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的东西

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，运动触发

### SSH

- home-server → 192.168.1.100, 用户：admin

### TTS

- 偏好语音："Nova"（温暖，略带英式）
- 默认扬声器：厨房 HomePod
```

## 为什么分开？

Skills 是共享的。你的设置是你的。分开它们意味着你可以更新 skills 而不丢失你的笔记，分享 skills 而不泄露你的基础设施。

---

添加任何能帮助你工作的东西。这是你的秘籍。
